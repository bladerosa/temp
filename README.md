# temp

CCPayment 内部原型仓库。

## 子项目

| 目录 | 项目 | 技术栈 |
|---|---|---|
| [`notifaction/`](./notifaction) | 通知系统 | React + Vite + TypeScript |
| [`collection/`](./collection)   | 归集系统  | React + Vite + TypeScript |

## 启动

```bash
# 通知系统
cd notifaction && npm install && npm run dev

# 归集系统
cd collection && npm install && npm run dev
```
